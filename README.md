# Implémentation d’un jeu de cartes (jeu_carte)

## Contexte
Les jeux de cartes reposent sur des piles (LIFO).

## Objectifs pédagogiques
- Piles
- Historique des actions
- Simulation de jeu

## Fonctionnalités
- Battre les cartes
- Piocher
- Poser une carte
- Consulter le sommet
- Vérifier si la pile est vide
- Annuler un coup
